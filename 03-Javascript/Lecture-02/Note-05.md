## Final Roadmap

| Chapter                                | Core Idea                                       | Why It Matters                                                                   |
| -------------------------------------- | ----------------------------------------------- | -------------------------------------------------------------------------------- |
| **4.1 Objects**                        | What objects are and how to create them         | Foundation of JavaScript data modeling                                           |
| **4.2 Object References & Copying**    | Objects are stored and copied by reference      | Prevents common bugs with mutation                                               |
| **4.3 Garbage Collection**             | How JavaScript manages memory                   | Helps avoid memory leaks and understand object lifetime                          |
| **4.4 Object Methods & `this`**        | Functions inside objects and context binding    | Essential for object-oriented programming and event handling                     |
| **4.5 Constructor Functions & `new`**  | Creating multiple similar objects               | Bridge to prototypes and ES6 classes                                             |
| **4.6 Optional Chaining (`?.`)**       | Safely access deeply nested properties          | Commonly used with APIs and optional data                                        |
| **4.7 Symbol Type**                    | Unique primitive values for special object keys | Enables hidden properties and advanced language features                         |
| **4.8 Object to Primitive Conversion** | How objects convert to strings and numbers      | Explains operators (`+`, `==`, template literals) and custom conversion behavior |
